Cubes

Cubes is a simple service designed to bootstrap and view experiments made with Three.js. It works by referencing code and files hosted on GitHub Gist. Simply code up a Three.js experiment and then reference cubes.heroku.com/[gist id] to allow others to view your experiment and source code!

Three.js bootstrapping

It is based on bl.ocks by the exellent Mike Bostock. However, this service is oppioniated and has the following convensions:

The file to render is called index.html. If you do not supply your own index.html then then a default, three.js-bootstrapped, index.html is used instead. The default file references the following files:
  • setup.js for setting up a three.js scene
  • script.js for your own code. Both of these files can be overwritten by your own files with the same name.
All files (i.e. index.html, setup.js and script.js) can be overwritten by your own files.
